C Project: Make and sell fun sandwiches 翻译：C 项目：制作和销售有趣的三明治

1 How many of these do they have? 翻译：1 他们有多少个？

Listen and circle. 翻译：听一听，圈一圈。

—Let's make money for wild elephants in Yunnan. 翻译：——让我们为云南的野生大象赚钱。

We can sell fun sandwiches. 翻译：我们可以卖有趣的三明治。

—Great idea! 翻译：——好主意！

—How many eggs do we have? 翻译：——我们有多少个鸡蛋？

—Hmm. We have twenty. 翻译：——嗯。我们有二十个。

—How about bananas and oranges? 翻译：——香蕉和橙子呢？

—One, two, ... thirteen, fourteen. 翻译：——一、二、⋯⋯十三、十四。

We have fourteen bananas. 翻译：我们有十四根香蕉。

One, two, ... fifteen, sixteen, seventeen, eighteen. 翻译：一、二、⋯⋯十五、十六、十七、十八。

We have eighteen oranges. 翻译：我们有十八个橙子。

—Good. 翻译：——很好。

We also need some grapes, bread and vegetables. 翻译：我们还需要一些葡萄、面包和蔬菜。

Let's go to the shop. 翻译：我们去商店吧。
